About the model

Since you are on this page, we will introduce the model we use for now.

We compared several models and chose U-Net model for now. U-Net is highly suitable for binary segmentation of images, which exactly matches our task: binary segmentation for whether pixels include a ship.

  • Training set includes 231,724 satellite images.
    • For images in the training set, the ground truth EncodedPixels are provided in the train_ship_segmentations.csv file.
    • EncodedPixels is blank if there are no ships in the image.

Validation loss (Intersection over Union): -0.25026
